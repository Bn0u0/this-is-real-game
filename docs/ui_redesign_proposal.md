# 主介面 UI 重構提案 (Main Menu Redesign Proposal)

針對《這才叫割草》的 "Cyberpunk / Wasteland" 風格與 "Mobile Action Roguelite" 的玩法，我們根據 2024-2025 的競品分析 (Brawl Stars, Zenless Zone Zero, Dead Cells)，提出三種截然不同的排版方案供您選擇。

所有方案皆遵循：**手機操作優先 (Thumb-friendly)**、**沉浸式背景 (Immersive Background)**、**明確的視覺層級 (Visual Hierarchy)**。

---

## 方案 A: The "Combat Hub" (經典戰鬥中樞)
**參考對象:** *Brawl Stars (荒野亂鬥), Apex Legends Mobile*

這是最安全且經過市場驗證的佈局。強調「角色展示」與「快速開局」。

### 佈局特徵
*   **中心舞台**: 3D 角色位於螢幕正中央 (或稍偏左/右)，可旋轉檢視。
*   **四周環繞**:
    *   **頂部**: 資源欄 (金幣、鑽石)、玩家等級/頭像、設置。
    *   **右側**: 「軍械庫 (Arsenal)」、「特工 (Agent)」、「商店」以垂直圖標排列。
    *   **左側**: 社交/任務/郵件 (較次要的功能)。
    *   **底部 (右下)**: 巨大的黃色/亮色 **[出擊 / DEPLOY]** 按鈕 (視覺焦點)。
*   **背景**: 3D 場景 (現在的 Workbench)，但 UI 浮在上面。

### 優點
*   ✅ **極低學習成本**: 手遊玩家最熟悉的佈局。
*   ✅ **強調角色**: 鼓勵玩家切換角色欣賞。
*   ✅ **手指熱區**: 主要功能都在拇指可及範圍。

### 缺點
*   ❌ **較為傳統**: 缺乏獨特的「驚艷感」。
*   ❌ **畫面易雜亂**: 如果功能按鈕多，畫面四周會被塞滿。

---

## 方案 B: The "Tactical Minimalist" (戰術極簡風)
**參考對象:** *Zenless Zone Zero (絕區零), Arknights (明日方舟), Nier: Automata*

強調「潮」、「風格化」與「電影感」。適合硬派 Cyberpunk 風格。

### 佈局特徵
*   **非對稱設計**: 3D 角色位於右側或左側，佔據 2/3 畫面。
*   **文字大於圖標**: 使用粗體、巨大的排版文字 (Typography) 作為按鈕。例如巨大的 "DEPLOY" 字樣橫跨背景。
*   **傾斜/故障藝術**: 選單使用傾斜角度 (Italic)，搭配 Glitch 效果。
*   **隱藏式 UI**: 次要功能 (設置、郵件) 收納在漢堡選單或角落的小點中，保持畫面極度乾淨。
*   **互動背景**: 背景會隨選單切換而運鏡 (Camera Pan)。

### 優點
*   ✅ **視覺衝擊力強**: 極具高級感與風格化，符合 "Vibe Coding" 精神。
*   ✅ **沉浸感**: 讓玩家感覺像是在操作一台高科技終端機，而不是玩遊戲。
*   ✅ **差異化**: 在眾多手遊中脫穎而出。

### 缺點
*   ❌ **易用性風險**: 純文字或風格化按鈕可能犧牲一點直覺性。
*   ❌ **開發成本**: 需要複雜的鏡頭運鏡與高品質的字體排版設計。

---

## 方案 C: The "Card Carousel" (卡片輪播式)
**參考對象:** *Marvel Snap, Clash Royale, 現代 Gacha 遊戲*

將主介面整合為可滑動的面板，適合強調「收集」與「備戰」的遊戲。

### 佈局特徵
*   **手勢操作**: 左右滑動切換不同「區域」(例如：主頁 <-> 角色 <-> 商店)。
*   **主頁即卡片**: 當前選中的角色以一張巨大的「卡片」或「展示櫃」形式呈現。
*   **底部導航欄 (Doc)**: 固定在底部的 4-5 個大圖標 (主頁、角色、軍械庫、商店)。
*   **出擊按鈕懸浮**: **[DEPLOY]** 按鈕懸浮在底部導航欄上方 (Floating Action Button style)。

### 優點
*   ✅ **超高擴充性**: 未來增加新系統 (如公會、寵物) 容易塞入。
*   ✅ **結構清晰**: 玩家永遠知道自己在哪個分頁。
*   ✅ **適合收集控**: 強調卡片與收集要素。

### 缺點
*   ❌ **沉浸感較弱**: 比較像是在操作 App，而不是置身於遊戲世界。
*   ❌ **背景被遮擋**: UI 佔比通常較大。

---

## 選擇建議

*   如果您想要 **最直覺、最快上手** -> 選擇 **方案 A (Combat Hub)**。
*   如果您想要 **最酷炫、最 Cyberpunk、最符合 "This is Real Game" 的氣質** -> 選擇 **方案 B (Tactical Minimalist)**。
*   如果您想要 **未來系統擴充方便** -> 選擇 **方案 C (Card Carousel)**。

**開發者推薦**: 考慮到您目前的 `WorkbenchScene` 已經有不錯的 3D 場景感，**方案 B (戰術極簡風)** 最能利用這個優勢，配合攝像機運鏡 (Camera Director) 創造出 3A 級的選單體驗。
